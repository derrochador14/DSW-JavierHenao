Fundamentos de HTML

¿Que es Html?

El HTML (HyperText Markup Language) es un lenguaje para describir el contenido de los documentos de la web. Utiliza una sintaxis especial que contiene marcadores (conocidos como "elementos") que rodean al texto que hay dentro del documento para indicar a los agentes de usuario cómo deben interpretar esta parte del documento.
segun Mark Norman Francis.

El html tambien utiliza una varia gama de señales y comando que ayudan a la interaccion, como titulos de diferentes niveles (h1,2,3,4) comando de parrafo (p) comandos de articulos, secciones, imagenes y tantas cosas que son casi universales.

cuando Tim Berners-Lee inventó la web tambien invento la primera version del html algunos comandos siguen completamente originales y otras han evolucionado incluso siendo exclusicas de algunos servidores
